# Debug e logging{#debugging-and-logging}

La registrazione del browser TVSDK è abilitata per impostazione predefinita. Fornisce inoltre il supporto API per abilitare o disabilitare la registrazione dall’SDK. Utilizza l’API AdobePSDK.LogManager.enableLogging per abilitare o disabilitare i registri.
